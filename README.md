# Behavioral Activation Tracker

A WIP desktop application focused on assisting individuals going through **Behavioral Activation**, a common treatment for the lack of motivation/task initiation caused mainly by Major Depressive Disorder (MDD), Persistent Depressive Disorder (PDD) or similar mental health conditions.

This project does not aim to be a substitute to formal psychotherapy, but merely an intuitive and useful tool to be used alongside it.
